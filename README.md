# image_filters_app
apply some basic filters to image with simple ui

# how to run app
	- source .venv/bin/activate
	- pip install streamlit pillow numpy scipy opencv-python
	- streamlit run app.py

Project Topic:
Apply all the filters studied during the course and demonstrate their effects through a GUI interface. Additionally, determine the best enhancement combination by showing step‑by‑step how to obtain a high‑quality image from a distorted input image.
Total degree: 20 marks – partitioned as follows:
* 5 marks – Idea & problem understanding
* 5 marks – GUI design & user interaction
* 5 marks – Implementation (correct application of filters & enhancement pipeline)
* 5 marks – Personal questions (viva voce)
Required deliverables:
* A working GUI application (Python/MATLAB/any language of your choice) that allows the user to: 
	• Load a distorted image
	• Apply each studied filter individually
	• View results side‑by‑side
* A step‑by‑step demonstration (within the GUI or as a separate report) of the best enhancement combination, showing how the distorted image is progressively improved.
* PowerPoint presentation illustrating your project idea, GUI design, implementation details, and the final enhancement pipeline.
* Project report (optional but recommended) summarizing the filters used, the chosen enhancement sequence, and experimental results.
